# Trapeze Issues

## Easily reproduce issues for [Trapeze](https://github.com/ionic-team/trapeze) with [CapacitorJS](https://capacitorjs.com/). 

### Steps

```shell
npm install

npm run build:ios
# or
npm run build:android

# apply trapeze config
npm run trapeze
```

### Running the webapp

To run the webapp, you can use `npm start` command.

```bash
npm start
```
